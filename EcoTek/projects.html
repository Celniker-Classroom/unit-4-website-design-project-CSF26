Waste Wise

Our 2025-2026 School Year Project

Waste Wise is a smart compost bin created to help schools reduce food waste and improve composting practices. By using sensors and engineering design, the project aims to monitor compost conditions and make the composting process more effective, efficient, and educational.

Materials and Uses

Material Use
Temperature Sensor Tracks the temperature inside the compost bin
Moisture Sensor Measures moisture levels to help maintain healthy compost
Gas Sensor Detects gases released during decomposition
Microcontroller Collects sensor data and helps run the system
Display or Output System Shows data and results from the compost bin
